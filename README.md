### Features

- add to cart (Provider)
- remove from cart (Provider)
- toggle favorite (Provider)
- List view builder
- APIs REST (GET)
- API from https://github.com/Re4ch-Jay/products_laravel_api (Custom API)
- Toast message
- Much more
